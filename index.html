DIY Spotify Wrapped 2022

December 2022 ∙ Python (plotly, spotipy, & more!), Tableau

This project is an analysis of my Spotify Premium account utilizing my streaming history and using Spotify's Web API. Python was used to access the data then clean and transform it. Space Missions

This project explores the Space Missions dataset from Maven Analytic’s Data Playground. SQL was utilized for data exploration and transformation. Tableau was then used to create an interactive dashboard to explore KPIs, space missions over time, rocket utilization, country success, and more!

NOAA Storm Events 2021

November 2022 ∙ Python (geopy, calplot, & more!)

This project explores the 2021 Storm Events Database published by the National Centers for Environmental Information's National Oceanic and Atmospheric Administration (NOAA). Recorded is data regarding the location, fatalities, and details of storms and other significant weather phenomena. Python was used to clean, transform, and visualize the data.
